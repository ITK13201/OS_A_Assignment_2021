# 課題1
## プログラムファイルのコンパイル方法
"splitfile.c"が置かれているディレクトリに移動する．コマンドライン上で"gcc -o splitfile splitfile.c"と打ち込むことでコンパイルできる．

## 実行時のコマンド
実行ファイル"splitfile"が置かれているディレクトリに移動する．コマンドライン上で"./ splitfile <入力ファイル名> <出力ファイル名1> <出力ファイル名2> <分割位置>"と打ち込むことで実行できる．
例えば，カレントディレクトリの下のexampleディレクトリにinput.txt，output1.txt，output2.txtが存在している場合，"./splitfile example/input.txt example/output1.txt example/output2.txt 10"とすればよい．
